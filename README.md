# SHLO

Shallow baseline models for text.

## Implemented models

* Linear model over pretrained embeddings
* fastText

## Getting started

### Install dependencies

* numpy
* tensorflow>=1.0

### Get data

```
./get_senna.sh
./get_polarity.sh
```

### Run tests for simple models

```
python shlo_test.py
```
